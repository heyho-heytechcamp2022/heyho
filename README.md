# heyho

## Setup

### Prepare

```bash
$ npm i
```

### Common Types

```bash
$ cd common/Types
$ npm i
```

### Firebase

```bash
$ npm i -g firebase-tools
$ cd firebase/functions
$ npm i
$ npm run serve  # launch local emulator
```

### Client

You should launch local emulator before `npm run dev`.

```bash
$ cd client
$ npm i
$ npm run dev
```
